TMC、TMG、TMD、TME、TMI 以 ISO 镜像文件方式提供。在 [TM 企业注册平台](https://www.TeloMatrixcloud.com/) 下载 ISO 安装文件，在虚拟机或物理机部署和激活。

注意：

1. 安装后先配置网络，再同步证书激活组件。激活需输入 TM 企业帐号和密码。

2. TMC 是核心组件，首次部署须先安装部署 TMC 并同步证书，再部署其他组件。

本文以虚拟机部署 TMC 为例。

警告：根据中国相关法律法规要求，大河乐境提供的 TM 安全云网产品组件（包括：TMC、TMG、TMD、TME、TMI）仅限在中国大陆区域部署使用。大河乐境已通过技术手段限制上述产品组件在中国大陆以外区域部署使用。使用方通过任何方法绕过此限制，所引起的相关后果与责任均由使用方承担，若此行为侵犯大河乐境的合法权益，大河乐境将依法追究使用方的法律责任。

## 第一步：获取安装介质

1. [注册 TM 企业帐号](How_to_Register_TM_Enterprise_Account)，安装过程中需使用该帐号激活组件。

2. 登录 [TM 企业注册平台](https://www.TeloMatrixcloud.com/)，进入【下载中心】下载安装 ISO 文件。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/1.Download_Installation_File.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

试用或永久免费权限可下载 TMC、TMG、TMD、TMA。TMPro 用户可下载所有组件。移动版 TMA 和 TM 令牌 APP 需扫码在应用商店下载。

## 第二步：安装 TM 组件

各组件可安装在虚拟机、物理服务器或云服务器上。以 VMware Fusion Player 部署 TMC 为例。物理服务器或其他虚拟机部署过程类似，公有云部署参考：[如何在公有云部署 TM](../2_Flexible_Usage/1_Installation_Deployment/How_to_Deploy_TM_on_Public_Cloud)。

## 第三步：组件初始化配置

安装完毕后登录组件控制台界面，完成网络基础配置和证书同步操作。

### 登录 TMC 控制台

重启虚拟机后自动显示 TMC 控制台界面（配置好网络后也可通过 SSH 登录）。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/19.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

输入用户名和密码，选择"登录"按回车进入控制台界面。

> TMv209x 版所有组件控制台默认用户名：admin，默认密码：b4a5928c

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/20.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

### 查看系统信息

选择"1"（系统信息）按回车，查看系统信息和 IP 地址分配情况。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/21.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

已分配 IP 地址则输入"q"返回，输入"4"进入证书同步。未分配 IP 地址则输入"q"返回，输入"2"先配置网络。

### 基础网络配置

- 在菜单页输入"2"（网络配置）按回车。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/22.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 输入"1"按回车进入接口配置界面。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/23.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 选择接口（输入"1"或接口名称如 eth0）。

- 选择接口模式：输入"dhcp"按回车使用 DHCP 自动分配。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/24.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 手动配置网络：输入"static"按回车，按提示输入 IP 地址、子网掩码、网关和 DNS 信息。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/25.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 在网络配置菜单界面输入"2"按回车进入 DNS 配置界面。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/26.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 按提示输入 DNS 配置信息，按任意键返回上级菜单完成网络配置。

### 同步授权证书

返回控制台菜单界面，输入"4"按回车进入同步证书界面。

- 首次部署，系统检测到本地无证书，出现输入 TM 云服务帐号和密码的界面，输入注册的帐号和密码。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/27.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 输入"u"按回车显示二维码，可扫码打开 TM 企业用户注册平台注册企业帐号，再用注册的帐号 ID 和密码登录。注册步骤参考：[如何注册 TM 企业帐号](How_to_Register_TM_Enterprise_Account)

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/28.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 曾经同步过证书，系统自动检测到本地存在证书并显示证书信息，输入"r"按回车重新加载和更新证书。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/29.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

完成以上步骤后 TMC 安装成功。

TMC 证书同步完成后，注册邮箱将收到 TMC 初始化邮件，包含 TMC 初始化帐号和密码。通过浏览器访问 TMC Web 管理控制界面（URL：https://IP地址:16000），使用初始化帐号和密码登录。

## 控制台其他功能

### 网络诊断

控制台主菜单输入"3"按回车进入网络诊断界面。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/30.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

#### ping 测试

测试 TM 组件网络连通性。输入"1"按回车进入 ping 测试界面。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/31.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

输入目标 IP 地址或域名，按回车发起 5 次 ping 探测。

#### traceroute 测试

测试 TM 组件到目标主机经过的路由器及耗时。输入"2"按回车进入测试界面。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/32.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

输入目标 IP 地址或域名，按回车发起 traceroute 探测。

#### curl 测试

测试 TM 组件与目标 Web 应用资源的连通性。输入"3"按回车进入测试界面。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/33.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

输入目标 URL，按回车发起 curl 请求，目标站点可正常访问时显示回显信息。

#### 查看网络状态

输入"4"按回车执行 netstat 指令，查看系统网络状态。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/34.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

#### 查看本地路由

输入"5"按回车执行 route 指令，查看本地路由状态。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/35.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

#### 查看本地网卡信息

输入"6"按回车执行 ip addr 指令，查看系统网卡状态。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/36.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

### 帐号管理

控制台主菜单输入"5"按回车进入帐号管理子菜单。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/37.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 输入"1"按回车修改控制台 admin 帐号密码。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/38.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/39.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- 输入"3"或"4"按回车分别启用或停用 Web 超级管理员帐号。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Deploy_TM_Components/40.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />
