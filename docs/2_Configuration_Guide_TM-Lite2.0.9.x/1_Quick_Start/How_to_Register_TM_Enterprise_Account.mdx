> 本文介绍新用户如何注册 TM 企业帐号，如何获取 TM 各组件的安装文件，以及如何申请 TMLite永久免费权限或升级到 TMPro权限。

## 注册 TM 企业帐号

企业用户可以在 TM 企业注册平台上注册企业帐号，并自行下载和部署 TM 安全云网各功能组件，使用 TM 安全云网服务。

TM 安全云网为每一个新注册企业用户提供了 TMLite的1个月免费试用授权。

注册 TM 帐号，需要您在[TM 企业注册平台](https://www.TeloMatrixcloud.com/#/login)登录页点击“立即注册”，填写以下信息，完成 TM 企业帐号注册：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Register_TM_Enterprise_Account/1.Register_Enterprise_Account.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**请您务必填写正确的管理员企业邮箱地址和管理员手机号**，我们将通过您填写的邮箱地址发送初始密码，通过您填写的手机号发送手机验证码。

注册成功后，您的帐号信息以及初始密码将通过邮件发送到您填写的企业邮箱，您可以根据邮件里的信息登陆 TM 云服务平台。

登录后，您可以查看到您当前的帐号状态以及授权状态，您可以在页面上提交申请，升级为永久免费授权或联系我们升级为 TMPro授权。如下图所示：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Register_TM_Enterprise_Account/2.Enterprise_Registration_Platform_Home.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />


## 获取 TMLite永久免费授权

TM 为中小企业提供永久免费的 TMLite版本，帮助中小企业解决：远程安全访问、互联网暴露面隐藏、在分布式网络里基于身份进行网络访问控制等安全和运维问题。

永久免费的 TMLite版本授权范围如下：

- 授权组件：

  TMC x 1、TMG x 5、TMD x 1、TMA无限制

- 授权范围：

  支持50个永久免费并发在线终端授权支持注册帐号数500个支持注册终端数1000个支持200个业务资源

  您可以在登录 TM 服务平台后，点击”申请永久免费使用“按钮提交企业认证信息，我们将在1个工作日内完成审核并将您的授权升级为永久免费授权。

## 申请 TMPro专业版授权

如果您的企业需要更多的 TM 安全云网并发终端数或者功能授权，您可以在 TM 服务平台提交 TM 专业版申请。

申请提交方式：登录 TM 企业注册平台后，在首页点击”申请专业版“，打开申请界面，您可以在描述里输入您的需求，然后点击”提交“即可。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Register_TM_Enterprise_Account/4.Apply_for_Pro_Version.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

在接收到您的申请1个工作日内，我们的客户经理会通过您注册时填写的手机号或邮箱与您取得联系，为您提供专业的技术指导和商务服务。
