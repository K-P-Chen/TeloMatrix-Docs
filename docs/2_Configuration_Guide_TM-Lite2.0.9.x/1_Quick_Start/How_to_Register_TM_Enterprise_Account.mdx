## 注册 TM 企业帐号

1. 在 [TM 企业注册平台](https://www.TeloMatrixcloud.com/#/login) 登录页点击"立即注册"。

2. 填写企业信息，完成注册。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Register_TM_Enterprise_Account/1.Register_Enterprise_Account.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

填写正确的管理员企业邮箱地址和手机号。初始密码通过邮箱发送，手机验证码通过手机号发送。

3. 注册成功后，帐号信息和初始密码通过邮件发送到企业邮箱。使用邮件信息登录 TM 云服务平台。

4. 登录后可查看帐号状态和授权状态，提交永久免费授权申请或升级为 TMPro 授权。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Register_TM_Enterprise_Account/2.Enterprise_Registration_Platform_Home.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

新注册企业用户享有 TMLite 1 个月免费试用授权。

## 获取 TMLite 永久免费授权

TMLite 永久免费授权范围：

- 授权组件：TMC x 1、TMG x 5、TMD x 1、TMA 无限制
- 授权范围：50 个永久免费并发在线终端、注册帐号数 500 个、注册终端数 1000 个、200 个业务资源

登录 TM 服务平台后，点击"申请永久免费使用"按钮提交企业认证信息。1 个工作日内完成审核并将授权升级为永久免费授权。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Register_TM_Enterprise_Account/3.Apply_for_Permanent_Free.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

## 申请 TMPro 专业版授权

如需更多并发终端数或功能授权，在 TM 服务平台提交专业版申请。

操作：登录 TM 企业注册平台，在首页点击"申请专业版"，填写需求后点击"提交"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/1_Quick_Start/How_to_Register_TM_Enterprise_Account/4.Apply_for_Pro_Version.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

客户经理在 1 个工作日内通过注册手机号或邮箱联系，提供技术指导和商务服务。
