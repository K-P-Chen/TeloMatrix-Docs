## TMG 列表

选择菜单【状态监控】->【云网监控】，点击【TM 网关】标签页：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

列表中展示 TMG 的管理状态、在线状态和服务状态。

- **管理状态：**管理员强制启停后的状态（正常/停用）。
- **在线状态：**TMG 与 TM 云服务及 TMC 的网络连通状态（在线/离线）。离线时 TMG 无法代理企业内网业务资源，需尽快恢复。
- **服务状态：**TMG 关键服务是否正常。
- **启用/停用：**勾选后点击列表功能区按钮可启用或停用 TMG。
- **注销：**点击"注销"可注销当前 TMG，证书变为"未激活"，所有配置和数据删除且不可恢复。注销后可重新部署，使用同一授权。

### TMG 代理资源

代理资源列显示当前 TMG 代理的业务资源数量，点击数字弹出代理资源列表：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/2.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

### TMG 组件状态

点击操作列"组件状态"按钮：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/3.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- **系统信息：**组件名称、序列号、云网管理 IP。
- **服务器状态：**CPU 利用率、内存利用率和存储空间利用率。
- **服务状态：**关键服务状态，异常时可手动重启。
- **实时速率：**近 24 小时流量转发速率。

#### 网络配置

点击"网络配置"标签页：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/12.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**接口信息：**查看 TMG 服务器网络接口配置，"业务接口"列显示"是"代表虚拟网接口。

**路由配置：**查看网络路由配置，点击"添加"手动添加路由：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/13.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

注意：只有手动添加的路由可删除，系统默认路由不可删除。

**网络出口：**TMG 部署时需在出口防火墙上配置 DNAT，将 TMG 物理 IP 的 UDP 9527 端口映射到出口公网 IP 的固定 UDP 端口。在此填写出口公网 IP 和端口：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/14.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**源 IP 地址转换：**默认开启，转发后的流量源 IP 转换为 TMG IP。关闭后源 IP 转换为终端用户的虚拟 IP，业务资源日志中可看到具体终端 IP。关闭后需在业务服务器上增加回指路由，将默认网关指向 TMG 内网 IP：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/15.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**流量采集（TMPro 功能）：**TMG 可采集终端用户访问业务资源的网络流量日志。选择需采集的协议：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/16.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**应用日志采集（TMPro 功能）：**TMG 内置 HTTP/HTTPS Web 应用代理和 SSH 代理。应用代理可采集加密内容解密后的字段（如 HTTPS 的 URL、Cookie，SSH 的指令和回显）。勾选需采集的协议，点击"保存"：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMG_Remote_Monitoring/17.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />
