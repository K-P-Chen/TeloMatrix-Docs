## TMC/TMI 列表

选择菜单【状态监控】->【云网监控】，点击【TM 智能分析平台】标签页查看 TMI 列表，点击【TM 管理控制平台】标签页查看 TMC 列表：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMC_and_TMI_Remote_Monitoring/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMC_and_TMI_Remote_Monitoring/2.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

列表中展示 TMC 或 TMI 的在线状态和服务状态。

- **在线状态：**TMC 与 TM 云服务、TMI 与 TM 云服务和 TMC 的网络连通状态，值为在线或离线。离线时终端用户无法通过 TMC 认证和同步策略，或 TMI 无法接收日志，需尽快恢复。
- **服务状态：**TMC 或 TMI 关键服务是否正常。
- **注销：**点击"注销"可注销当前组件。证书将从"激活"变为"未激活"，组件退出云网，所有配置和数据删除且不可恢复。注销后可重新部署并用同一授权激活，不额外占用授权。

## TMC 组件状态

点击 TMC 列表中的"组件状态"：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMC_and_TMI_Remote_Monitoring/3.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- **系统信息：**组件名称、序列号、云网管理 IP 等。可点击"修改"按钮修改组件名称。
- **服务器状态：**CPU 利用率、内存利用率、存储空间利用率和日志存储总量。
- **服务状态：**关键服务状态，异常时可手动重启。
- **认证请求趋势：**近 24 小时认证请求趋势。

## TMI 组件状态

点击 TMI 列表中的"组件状态"：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMC_and_TMI_Remote_Monitoring/4.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- **系统信息：**组件名称、序列号、云网管理 IP 等。
- **数据同步配置：**TMI 与 TMC 同步数据的 IP 地址（通常填写 TMI 物理 IP，需与 TMC 互通）和端口（9092）。
- **服务器状态：**CPU 利用率、内存利用率、存储空间利用率和日志存储总量。
- **服务状态：**关键服务状态，异常时可手动重启。
- **数据采集趋势：**近 24 小时数据采集量趋势。
