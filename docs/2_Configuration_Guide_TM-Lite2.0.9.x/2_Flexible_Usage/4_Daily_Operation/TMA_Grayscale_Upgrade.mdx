## 功能说明

TM 安全云网支持对网络内终端用户使用的TMA进行灰度升级，即当TMC检测到有新版本的TMA可以升级的时候，可以指定一部分终端用户先升级体验，然后再升级其他终端用户。

## TMA灰度升级操作方法

1. 选择菜单【系统管理】->【系统维护】->【客户端维护】打开客户端升级维护页面，如下图所示：

 <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMA_Grayscale_Upgrade/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 客户端下载地址

  您可以直接访问页面上的下载地址，下载最新的TMA客户端软件，终端用户在自动升级TMA客户端软件时也是通过此地址下载最新的安装文件。

  如果您使用的是定制版本的TMA客户端软件，您可能需要修改此下载地址，具体的下载地址请联系我们获得。

3. 可升级客户端列表

  如果有新的TMA版本可以升级，TM 将推送可升级的版本信息到此列表，您可以将升级包推送给指定的部分终端用户，或强制这些终端用户进行升级。

4. 在可升级客户端列表选择要升级的版本，点击【绑定帐号】可进行软件包推送，如下图所示：

  <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMA_Grayscale_Upgrade/2.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

  **强制升级：**当选择强制升级后，终端用户在下次打开TMA的时候，用户必须先升级然后才可以认证登录。

  <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMA_Grayscale_Upgrade/3.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

  **推送升级：**当选择推送升级后，终端用户在下次打开TMA的时候，可以选择是否立即更新，如果选择暂不更新，则会直接进入认证登录界面，在下一次打开TMA时再次提示升级。

  <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMA_Grayscale_Upgrade/4.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />
