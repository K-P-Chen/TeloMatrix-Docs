TME 为 TMPro 功能。

## TME 列表

选择菜单【状态监控】->【云网监控】，点击【TM 边缘网关】标签页：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

列表中展示 TME 的管理状态、在线状态和服务状态。

- **管理状态：**管理员强制启停后的状态（正常/停用）。
- **在线状态：**TME 与 TM 云服务及 TMC 的网络连通状态（在线/离线）。离线时 TME 无法代理企业内网业务资源，需尽快恢复。
- **服务状态：**TME 关键服务是否正常。
- **启用/停用：**勾选后点击列表功能区按钮可启用或停用 TME。停用后管理状态变更为"停用"。
- **注销：**点击"注销"可注销当前 TME，证书变为"未激活"，所有配置和数据删除且不可恢复。注销后可重新部署，使用同一授权。

代理资源列显示当前 TME 代理的业务资源数量，点击数字弹出代理资源列表：

## TME 组件状态

点击操作列"组件状态"按钮：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/3.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- **系统信息：**组件名称、序列号、云网管理 IP。
- **服务器状态：**CPU 利用率、内存利用率和存储空间利用率。
- **服务状态：**关键服务状态，异常时可手动重启。
- **实时速率：**近 24 小时流量转发速率。

### 基础信息

**系统信息：**点击"修改名称"可修改 TME 组件名称：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/5.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**门户配置：**TME 为终端用户提供统一 Web 应用代理门户，终端用户无需 TMA 即可通过 TME 进行身份认证和业务访问，支持 HTTP、HTTPS、SSH、RDP 协议：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/6.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/7.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

在门户配置区可自定义背景图、Logo 等信息：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/8.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- **域名：**终端用户通过域名访问 TME 时填写。
- **端口：**TME 门户 Web 服务开放端口。
- **认证方式：**支持帐号密码（TMC 本地/AD/LDAP）和扫码（钉钉/企业微信/飞书/TMA）登录。
- **证书：**选择已上传到 TMC 的 SSL 证书实现加密访问。
- **登录页背景图：**建议尺寸 520x635，不超过 500KB，支持 jpg/png。
- **登录页 Logo：**建议尺寸 140x40，不超过 500KB，支持 jpg/png。
- **版权信息：**最多 200 字符。

**时间设置：**设置时区、NTP 服务地址和同步周期（1-720 小时）：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/9.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**网络测试：**TME 需保持物理网络可达所有代理或转发的业务资源。支持 ping、trace、ifconfig、tcping 四种测试命令：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/10.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/11.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/13.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**系统维护：**手动重启 TME 或恢复出厂设置。恢复出厂设置删除所有配置但保留授权证书：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/14.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

注意：TME 代理业务资源时，重启或恢复出厂设置会导致终端无法访问被代理资源。操作前请确认不影响业务。

### 网络配置

点击"网络配置"标签页：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/15.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**接口信息：**查看 TME 服务器网络接口配置，"业务接口"列显示"是"代表虚拟网接口。

**路由配置：**查看网络路由配置，点击"添加"手动添加路由：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TME_Remote_Monitoring/16.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

注意：只有手动添加的路由可删除，系统默认路由不可删除。
