## TMD 列表

选择菜单【状态监控】->【云网监控】，点击【TMDNS】标签页：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMD_Remote_Monitoring/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

列表中展示 TMD 的管理状态、在线状态和服务状态。

- **管理状态：**管理员强制启停后的状态（正常/停用）。
- **在线状态：**TMD 与 TM 云服务及 TMC 的网络连通状态（在线/离线）。离线时 TMD 无法提供域名解析服务，需尽快恢复。
- **服务状态：**TMD 关键服务是否正常。
- **启用/停用：**勾选后点击列表功能区按钮可启用或停用 TMD。
- **注销：**点击"注销"可注销当前 TMD，证书变为"未激活"，所有配置和数据删除且不可恢复。注销后可重新部署，使用同一授权。

## TMD 组件状态

点击操作列"组件状态"按钮：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMD_Remote_Monitoring/2.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- **系统信息：**组件名称、序列号、云网管理 IP。
- **服务器状态：**CPU 利用率、内存利用率和存储空间利用率。
- **服务状态：**关键服务状态，异常时可手动重启。
- **寻址次数：**近 24 小时 DNS 寻址次数统计。

### 解析配置

点击"解析配置"标签页：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMD_Remote_Monitoring/7.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

TMD 默认自动生成域名代理业务资源的解析策略，也可手动创建。点击"新增"：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TMD_Remote_Monitoring/8.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />
