## 在线帐号监控

选择菜单【状态监控】->【在线资源】打开在线资源监控页面：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/1.png" style={{width: "100%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

页面上方分类统计区显示在线帐号数、在线终端数和在线业务资源数。点击各分类，下方趋势统计和明细列表显示对应详细信息。

### 在线帐号趋势统计

选中在线帐号后，趋势统计区默认显示近 24 小时在线帐号趋势。通过右上角统计时间选项可调整时间范围：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/2.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/3.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

### 在线帐号-终端列表

显示当前在线的帐号终端明细：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/4.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

列表总行数可能大于在线帐号数。每一行代表一组帐号与终端的组合（即一个访问源实体），同一个帐号在多台终端登录会产生多行。

### 帐号详情

点击列表中帐号 ID，查看帐号详情：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/5.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

帐号信息区显示姓名、手机号、邮箱、隶属帐号组、角色、有效期等属性。授权终端列表显示帐号当前授权的终端，点击"取消授权"可解除帐号与终端的授权关系。解除后该帐号再次使用此终端登录时，TMC 会要求重新注册终端。

注意：TME 统一由管理员在授权策略中绑定，此处不可取消授权。

### 强制下线

勾选帐号终端后，点击"强制下线"按钮可手动将帐号退出 TM 安全云网并封禁：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/6.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

## 在线终端监控

### 在线终端趋势统计

选中在线终端后，趋势统计区默认显示近 24 小时在线终端趋势。通过右上角统计时间选项可调整时间范围：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/7.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/8.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

### 在线终端列表

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/9.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

注意：终端 IP 地址为终端所在网络的公网 IP。

### 终端详情

点击列表中的终端名称查看详情：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/10.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

终端信息区显示类型、用途、终端组、责任人、终端ID、系统版本等。授权帐号列表显示可使用此终端的帐号，点击"取消授权"可解除绑定。

### 强制下线

勾选终端后点击"强制下线"可手动将终端退出并封禁：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/Online_Resource_Monitoring/11.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />
