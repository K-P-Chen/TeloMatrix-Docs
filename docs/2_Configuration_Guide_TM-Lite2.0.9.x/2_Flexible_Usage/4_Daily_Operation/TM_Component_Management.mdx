TMC作为 TM 安全云网的集中管理控制平台，可以对云网内所有TMG、TMD、TME进行集中管控。

当TMG、TMD、TME在服务器或虚拟机上部署后，管理员通过SSH接入系统后台管理界面并执行更新证书操作，TM 组件将自动接入到对应的虚拟安全云网，自动纳入网内TMC的管控。

TMI有独立管理界面，需单独进行管理。

## 云网状态监控

选择菜单【状态监控】->【云网监控】打开云网监控页面，如下图所示：

在页面上方，有三个信息区，显示您的 TM 安全云网授权和当前使用量，如下图所示：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TM_Component_Management/2.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />


**授权有效期：**

显示您的 TM 安全云网授权到期时间以及当前您已经使用了多长时间。

如果您的 TM 安全云网是试用授权，当授权到期后您将无法登陆TMC，您的 TM 安全云网也将停止服务；

如果您的 TM 安全云网是 TMLite永久免费授权，则到期时间将显示“永久免费”；

如果您的 TM 安全云网是 TMPro授权，当授权到期后您的 TM 安全云网将降级为永久免费授权，您依然可以使用您的 TM 安全云网，但授权并发终端将被限制为最多50个。

**并发终端数：**

显示您当前授权的最大并发终端数量，以及近30天最大并发终端数。

当您发现近30天的并发数已经快要达到您的授权最大并发终端数时，请及时与我们联系升级您的授权，以免影响终端用户的业务访问。

**代理业务资源数：**

显示您当前授权的最大业务资源数，TMLite版本最大授权代理的业务资源数为200个，TMPro版本最大授权代理的业务资源数为40000个。
