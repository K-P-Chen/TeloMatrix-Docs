TMC 作为 TM 安全云网的集中管理控制平台，可对云网内所有 TMG、TMD、TME 进行集中管控。组件部署后通过 SSH 执行更新证书操作，即可自动接入虚拟安全云网并纳入 TMC 管控。TMI 有独立管理界面，需单独管理。

## 云网状态监控

选择菜单【状态监控】->【云网监控】：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TM_Component_Management/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

页面上方有三个信息区显示授权和使用量：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/4_Daily_Operation/TM_Component_Management/2.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**授权有效期：**显示到期时间和已使用时长。试用授权到期后无法登录 TMC 且停止服务；TMLite 永久免费授权显示"永久免费"；TMPro 授权到期后降级为永久免费，并发终端限制为最多 50 个。

**并发终端数：**显示最大授权并发终端数和近 30 天最大并发数。临近上限时请及时联系升级授权。

**代理业务资源数：**显示最大授权业务资源数。TMLite 上限 200 个，TMPro 上限 40000 个。
