## 问题说明

企业 Web 业务资源中常出现页面跳转、重定向或嵌套：

1. 单点登录系统（SSO）场景：目标站点返回 302 跳转，要求浏览器先到 SSO 站点认证。
2. 业务资源调整时使用 301 重定向，将访问目标永久改为其他站点。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/1.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 页面嵌套：如内部门户将多个业务系统消息整合到一个页面。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/2.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

上述情况意味着访问一个 Web 业务资源时实际同时在访问多个站点。需要将跳转站点与业务资源关联授权，否则终端用户访问时会出现站点不全或页面无法显示的问题。

关联方法有两种：

a. 将业务资源放到同一个业务资源组，按资源组授权。
b. 配置业务资源代理时进行站点关联。

## 配置方法1：设置业务资源组（推荐）

将业务资源与关联资源放入同一资源组，授权时按资源组授权。

1. 选择菜单【资源管理】->【业务资源】，在资源组区点击"新增"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/3.png" style={{width: "30%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/4.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 勾选有关联的多个业务资源，点击"编辑"，批量编辑，将这些资源加入到新建的资源组。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/5.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 选择菜单【策略管理】->【授权策略】，点击"新增"，在资源选择区域选择新建的资源组，保存完成批量授权。

## 配置方法2：TMG自动关联

发布 Web 业务资源时，TMG 自动检测首页是否存在 301 或 302 跳转/重定向。

1. 选择菜单【资源管理】->【业务资源】，新建 Web 业务资源后点击"服务代理"。
2. 选择资源服务并开启网关代理时，网关自动检测。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/6.png" style={{width: "100%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 检测到跳转或重定向：
   - 如目标站点未发布到 TM，提示需先发布目标站点再关联。

   <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/7.png" style={{width: "40%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

   - 如目标站点已发布，提示是否立即关联。点击"关联"自动绑定。

   <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/8.png" style={{width: "40%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

   关联后在已关联跳转资源列表中可查看。

   <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/9.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

   如点击"不关联"，可在关联资源界面点击"手动关联"随时绑定。

   <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Page_Redirect_or_Embedding/10.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

注意：TMG 只能检测首页的跳转/重定向，无法检测其他页面的跳转和页面嵌套。如存在以上情况，建议使用配置方法1（设置业务资源组）。
