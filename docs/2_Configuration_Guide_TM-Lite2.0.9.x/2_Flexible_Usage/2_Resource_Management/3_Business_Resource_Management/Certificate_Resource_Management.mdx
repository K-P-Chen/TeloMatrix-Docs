---

证书管理功能用于统一管理 Web 应用资源的 SSL 服务端证书，发布 HTTPS 协议的 Web 应用资源时可引用这些证书。

## 新增证书

选择菜单【资源管理】->【证书资源】，点击"新增"按钮，完成配置后点击"保存"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Certificate_Resource_Management/1.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- **证书名称：**必填，不超过 100 个字符，中英文、数字、@.-_ 组合。
- **证书有效期：**到期后 TMC 自动提醒。留空则视为永久有效。
- **上传证书：**将 .key 和 .crt 文件打包成 .zip 压缩文件后拖拽至上传区域，或点击"点击上传"选择本地文件。

## 编辑证书

在证书资源管理页面点击"编辑"按钮，修改后点击"保存"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Certificate_Resource_Management/2.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

证书名称和有效期均可修改。证书文件替换后，所有引用此证书的业务资源同步更新。

## 查看证书

在证书资源管理页面点击证书名称，可查看引用此证书的业务资源列表。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Certificate_Resource_Management/3.png" style={{width: "40%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

注：访问客体为 TMPro 版本横向业务隔离功能，TMLite 版本可忽略。
