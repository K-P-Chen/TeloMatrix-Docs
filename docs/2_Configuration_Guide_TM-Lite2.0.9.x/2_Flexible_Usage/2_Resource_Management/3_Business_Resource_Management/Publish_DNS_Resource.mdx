title: "发布DNS资源"
---

DNS 资源用于为终端配置系统 DNS 参数。当授权 DNS 资源给帐号终端后，终端根据指定的 DNS 服务器进行域名解析，其他解析方式失效。DNS 资源在多种解析方式中优先级最高。

选择菜单【资源管理】->【业务资源】，点击"新增"按钮。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/3_Business_Resource_Management/Publish_DNS_Resource/1.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

填入基本信息，在"资源类别"选择"DNS资源"。

- **公网DNS：**选择"DNS类型"为"公网DNS"，在"DNS地址"填入公网 DNS 的 IP 地址。
- **内网DNS：**选择"DNS类型"为"内网DNS"，填入内网 DNS IP 地址后，还需选择代理内网 DNS 的 TMG。

关于内网与公网 DNS 的使用方法，参考[云网域名解析方案](../../../2_Flexible_Usage/1_Installation_Deployment/Cloud_Network_Domain_Resolution_Solution)。
