## 功能说明

1. 支持同时对接一个或多个 AD/LDAP 身份源。
2. 支持手动同步或定时同步 AD/LDAP 帐号信息。
3. 每个 AD/LDAP 身份源可配置不同的认证策略和访问权限。
4. 终端用户在 TMA 上使用 AD/LDAP 帐号登录时，使用对应身份源的认证服务进行验证。

## 新建AD/LDAP身份源

1. 选择菜单【策略管理】->【认证策略】，选择"认证接口"标签页。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_AD_or_LDAP_Identity_Source/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 在左侧选择"AD/LDAP认证"。
3. 点击![新增](/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_AD_or_LDAP_Identity_Source/2.png)按钮。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_AD_or_LDAP_Identity_Source/3.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

**AD 身份源配置项**

- **服务名称：**必填，不超过 100 个字符，中英文、数字、@.-_ 组合。
- **服务器类型：**选择"AD"。
- **服务地址：**AD 服务器 IP 地址，TMC 需能正常访问。
- **服务端口：**默认 389。
- **管理员帐号：**格式如 CN=Administrator。
- **管理员密码：**对应管理员密码。
- **Base DN：**格式如 DC=xx,DC=xx。
- **帐号过滤条件：**默认 (objectCategory=person)。
- **指定帐号字段：**默认 sAMAccountName。
- **指定姓名字段：**默认 cn。
- **指定描述字段：**默认 description。
- **指定手机号字段：**默认 telephoneNumber。
- **指定邮箱字段：**默认 mail。
- **指定角色字段：**非必填，指定作为角色字段的 AD 属性值。
- **备注：**非必填，最多 200 个字符。

**LDAP 身份源配置项**

- **服务名称：**必填，不超过 100 个字符，中英文、数字、@.-_ 组合。
- **服务器类型：**选择"LDAP"。
- **服务地址：**LDAP 服务器 IP 地址。
- **服务端口：**默认 389。
- **管理员帐号：**格式如 CN=Administrator。
- **管理员密码：**对应管理员密码。
- **Base DN：**格式如 DC=my-domain,DC=com。
- **帐号过滤条件：**默认 (objectCategory=person)。
- **指定帐号字段：**默认 uid。
- **指定姓名字段：**默认 cn。
- **指定描述字段：**默认 description。
- **指定手机号字段：**默认 telephoneNumber。
- **指定邮箱字段：**默认 mail。
- **指定角色字段：**非必填。
- **指定帐号全路径字段：**默认 entryDN。
- **备注：**非必填，最多 200 个字符。

4. 点击"保存"，AD/LDAP 服务被保存并启用。
5. 在服务列表点击"编辑"可修改配置。
6. 点击"清空数据"可清空此服务同步的所有帐号数据，此操作不可逆。
7. 选中服务后可使用列表功能区的"删除"、"启用"、"停用"按钮。

## 设置AD/LDAP帐号同步策略

1. 选择菜单【资源管理】->【帐号资源】，点击"帐号同步"按钮。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_AD_or_LDAP_Identity_Source/4.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 在 AD/LDAP 标签页，点击"同步设置"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_AD_or_LDAP_Identity_Source/5.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

- **帐号存储字符转换：**Windows 对大小写不敏感，TMC 对大小写敏感。选项："不转换"、"转小写"（默认）、"转大写"。
- **是否发送初始化邮件：**开启则同步时对首次帐号自动发送；关闭则需手动发送。
- **开启自动同步设置：**关闭则手动同步；开启则设置间隔分钟数或定时同步。

3. 选中服务后点击"开始同步"可立即同步。
4. 如需配置帐号密码认证优先级：选择【策略管理】->【全局策略】，在"终端全局配置"区设置"优先AD帐号认证"。启用则优先 AD/LDAP 认证，关闭则优先 TMC 本地认证。
