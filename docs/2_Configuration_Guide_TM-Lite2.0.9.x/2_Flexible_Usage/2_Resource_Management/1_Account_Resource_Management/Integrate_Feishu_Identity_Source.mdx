## 功能说明

1. TM 支持对接飞书身份源，支持PC和Mac终端用户使用手机飞书扫描TMA上的二维码进行身份验证和授权登录，支持终端用户在TME上使用手机飞书扫描TME上的二维码进行认证和授权登录。

2. 支持用飞书帐号设置用户的身份验证策略、业务资源授权策略、细粒度的网络访问控制策略，以及作为网络流量身份化的身份依据。

3. TMC支持手动或自动同步飞书身份源的帐号信息，当使用飞书身份源登录 TM 安全云网时，终端用户将使用飞书的认证服务进行身份认证。

4. 飞书目前不支持手机上的其他应用通过飞书APP进行关联认证，因此如果您使用飞书身份源，您的 TM 手机端APP无法使用已登录的飞书APP进行关联认证。（目前仅企业微信可以支持）

## 飞书服务端配置

在TMC上对接飞书身份源，需要先在飞书上进行授权配置，配置步骤如下：

1. 使用管理员帐号登陆飞书管理后台（打开[飞书后台登陆页面](https://www.feishu.cn/admin)，使用飞书管理员的飞书APP扫描二维码即可登录）。

2. 点击【我的后台】->【开发者后台】菜单，在首页点击“创建应用”创建一个专门用于与TMC对接的应用，如下图所示：

  <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/1.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

  点击“确定创建”完成应用的创建。

3. 在“开发者后台”页面，可以查看到刚才创建的应用，点击应用名称，进入应用编辑页面。点击“凭证与基础信息”获取“App ID”与“App Secret”应用凭证，如下图所示：

  <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/2.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

  请记录这两个应用凭证的值，它们将在TMC上配置飞书身份源接口时被用到。

4. 在应用配置页面，点击【安全设置】，填入重定向URL信息，重定向URL信息即飞书认证结果需要返回的地址，您可以在TMC的【策略管理】->【认证策略】->【认证接口】->【扫码认证】->【飞书】配置页找到分配给您的回调地址，如下图所示：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/3.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

5. 在应用配置页面，点击【权限管理】，为新建应用配置权限，如下图所示：

  <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/4.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

  需要开通的权限包括：

    - 获取用户 user ID
   
    - 获取用户手机号
  
    - 获取用户邮箱信息
  
    - 获取部门组织架构信息
  
    - 获取部门基础信息
  
    - 以应用身份读取通讯录
  
    - 获取用户基本信息

6. 在应用配置页面，点击【应用功能】->【移动应用登录】，开启飞书登录开关，选择用户登录协议为“OAuth2.0”，勾选应用类型包括iOS应用和Android应用。

  在“Bundle ID”参数里填写固定值“**com.yiYun.appClient**”；

  在“Android签名”参数里填写固定值“**1f20c88c1f71f71517cc1e7d60c43279**”；

  在“Android包名称”里填写固定值“**com.yjx.flutter_yjx_trust**”。

  如下图所示：

7. 在应用配置页面，点击【应用发布】->【版本管理与发布】，打开应用发布配置页面，点击“创建版本”按钮，为新建的应用创建版本，并申请发布，如下图所示：

  <img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/7.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

8. 点击飞书开放平台的平台菜单按钮，在平台菜单中点击“管理后台”图标，进入飞书后台管理界面，如下图所示：

9. 在飞书管理后台点击菜单【工作台】->【应用管理】，找到刚才发布的应用，并点击“配置”按钮，进入应用配置页面，如下图所示：

## 新建飞书身份源

1. 选择菜单【策略管理】->【认证策略】打开认证策略管理页面，在页面顶部选择“认证接口”标签页，显示TMC的认证接口配置页面；

2. 在认证接口配置页面选择“扫码认证”，并选择“飞书”标签页，如下图所示：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/12.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 在飞书接口配置项里，将在飞书管理后台记录的“App ID”、APP Secret”参数值填入，并点击“保存”按钮保存所有配置。

## 设置飞书帐号同步策略

1. 选择菜单【资源管理】->【帐号资源】打开帐号管理页面，在“帐号管理”标签页里点击“帐号同步”按钮，打开帐号同步策略配置抽屉页，选择“飞书”标签页，如下图所示：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/13.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 选择是否自动发送初始化邮件
  
  如果开启，则同步飞书帐号时，TMC将自动发送帐号初始化邮件到首次同步获取帐号的注册邮箱中。
  
  如果关闭，则同步飞书帐号时，TMC将不会给任何帐号自动发送帐号初始化邮件，您需要在帐号列表上选择需要发送的帐号，手动单独或批量为飞书身份源的帐号发送帐号初始化邮件。

3. 同步方式

  如果选择“手动同步”，则每次同步飞书帐号时，您都需要点击“立即同步”按钮，手动地同步帐号信息。

  如果选择“自动同步”，则您可以设置每隔多少分钟，或者在每天什么时刻、每周什么时刻，TMC自动地同步飞书帐号信息。

4. 清空数据

  清空已经从飞书身份源同步到的所有帐号和帐号组信息。

完成以上配置后，点击“保存并立即同步”完成飞书帐号同步策略配置。
