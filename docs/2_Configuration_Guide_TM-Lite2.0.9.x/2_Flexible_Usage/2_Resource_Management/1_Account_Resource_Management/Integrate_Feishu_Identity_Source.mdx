## 功能说明

1. 支持 PC 和 Mac 终端用户使用手机飞书扫描 TMA/TME 上的二维码进行身份验证和授权登录。
2. 支持使用飞书帐号设置身份验证策略、资源授权策略、访问控制策略以及网络流量身份化。
3. TMC 支持手动或自动同步飞书帐号信息，终端用户使用飞书认证服务进行身份验证。
4. 飞书不支持手机端其他应用通过飞书 APP 关联认证，TM 手机端 APP 无法使用已登录的飞书 APP 进行关联认证。

## 飞书服务端配置

1. 使用管理员帐号登录[飞书管理后台](https://www.feishu.cn/admin)。
2. 点击【我的后台】->【开发者后台】，点击"创建应用"创建一个用于与 TMC 对接的应用。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/1.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 在"开发者后台"点击应用名称进入编辑页，点击"凭证与基础信息"获取"App ID"与"App Secret"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/2.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

4. 点击【安全设置】，填入重定向 URL。回调地址在 TMC 的【策略管理】->【认证策略】->【认证接口】->【扫码认证】->【飞书】配置页获取。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/3.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

## 新建飞书身份源

1. 选择菜单【策略管理】->【认证策略】，选择"认证接口"标签页。
2. 选择"扫码认证"->"飞书"标签页。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/12.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 填入"App ID"、"APP Secret"，点击"保存"。

## 设置飞书帐号同步策略

1. 选择菜单【资源管理】->【帐号资源】，点击"帐号同步"按钮，选择"飞书"标签页。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_Feishu_Identity_Source/13.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 是否自动发送初始化邮件：开启则同步时自动发送到首次同步的帐号邮箱；关闭则需手动发送。
3. 同步方式：手动同步需每次点击"立即同步"；自动同步可设置间隔分钟数或定时同步。
4. 清空数据：清空从飞书同步的所有帐号和帐号组信息。

完成配置后，点击"保存并立即同步"。
