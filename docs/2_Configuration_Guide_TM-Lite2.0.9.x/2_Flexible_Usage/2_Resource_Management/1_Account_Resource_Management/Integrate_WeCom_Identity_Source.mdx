## 功能说明

1. 支持 PC/Mac 终端用户使用企业微信扫描 TMA/TME 二维码进行认证登录，移动端 TMA 可通过已登录企业微信进行关联认证。
2. 支持使用企业微信帐号设置身份验证策略、资源授权策略、访问控制策略以及网络流量身份化。
3. TMC 支持手动或自动同步企业微信帐号信息，终端用户使用企业微信认证服务进行身份验证。

## 企业微信服务端配置

1. 使用管理员帐号登录[企业微信管理后台](https://work.weixin.qq.com/wework_admin/loginpage_wx?from=myhome)。
2. 点击【应用管理】，在"自建"区域点击"创建应用"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/1.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 填写应用名称、介绍，上传 Logo，选择可见范围。仅选中范围内成员可在 TMA 上使用企业微信登录。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/2.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

4. 进入新应用页面，记录"AgentId"和"Secret"值。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/3.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

5. 在页底"开发者接口"区域，点击"企业微信授权登录"上的"设置"按钮。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/4.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

6. 选择"Web网页"，点击"设置授权回调域"，填入回调地址并保存。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/5.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

回调地址在 TMC 的【策略管理】->【认证策略】->【认证接口】->【扫码认证】->【企业微信】配置页获取，"重定向地址"即回调地址。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/6.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

iOS Bundle ID 填写固定值 **com.yiYun.appClient**；Android 应用签名填写 **1f20c88c1f71f71517cc1e7d60c43279**，应用包名填写 **com.yjx.flutter_yjx_trust**。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/7.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

7. 记录自动生成的应用程序 Schema。
8. 在【我的企业】->【企业信息】底端查看"企业ID"值。
9. 配置企业可信IP：在"开发者接口"区域点击"企业可信IP"的"配置"链接，填入 TMC 所在网络的公网 IP 地址。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/10.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/11.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

建议将 TMC 部署在使用静态公网 IP 地址的网络中，避免动态 IP 变更后需频繁修改此配置。

## 新建企业微信身份源

1. 选择菜单【策略管理】->【认证策略】，选择"认证接口"标签页。
2. 选择"扫码认证"->"企业微信"标签页。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/8.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 填入"企业ID"、"AgentID"、"Secret"、"Schema"，点击"保存"。

## 设置企业微信帐号同步策略

1. 选择菜单【资源管理】->【帐号资源】，点击"帐号同步"按钮。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_WeCom_Identity_Source/9.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 是否自动发送初始化邮件：开启则同步时自动发送到首次同步的帐号邮箱；关闭则需手动发送。
3. 同步方式：手动同步需每次点击"立即同步"；自动同步可设置间隔分钟数或定时同步。
4. 清空数据：清空从企业微信同步的所有帐号和帐号组信息。

完成配置后，点击"保存并立即同步"。
