title: "TM 帐号分类"
---

TM 安全云网中的帐号分为三类：帐号、主帐号、业务帐号。

## 帐号

帐号是员工在不同身份源（如企业微信、AD、钉钉）中分配的帐号，一个员工可拥有多个身份源的帐号。

**用途：**
- 通过各身份源的认证服务对终端用户进行身份验证。
- 对帐号进行资源授权，生成基于身份的访问控制策略。
- 以帐号为维度进行网络数据包身份识别和流量分析。

**约束：**支持本地身份源、钉钉、企业微信、飞书、AD/LDAP、CA 等身份源。本地身份源帐号支持增删改查；第三方身份源帐号仅支持查询，以及编辑有效时间和备注。

## 主帐号

主帐号是自动聚合不同身份源帐号生成的虚拟帐号，一个主帐号对应一个自然人。聚合通过帐号的姓名、手机号、邮箱地址自动关联。

**用途：**主帐号用于自然人的统一授权。授权主帐号后，其关联的所有身份源帐号同步获得授权。

**约束：**主帐号由系统自动生成，不可增删改。属性值不可编辑，但可从关联的某个身份源自动读取赋值，用于批量授权。

## 业务帐号

业务帐号是业务系统自带的帐号体系中的帐号，如 OA 系统的 admin 帐号、Linux 服务器的 root 帐号。

**用途：**默认用于登录业务资源系统。企业可通过单点登录系统统一接管多个业务系统的认证和授权。

**约束：**TM 与单点登录系统对接为 TMPro 版本功能，支持 CAS、Oauth2、OIDC、SAML 等协议。TMLite 版本暂不支持。
