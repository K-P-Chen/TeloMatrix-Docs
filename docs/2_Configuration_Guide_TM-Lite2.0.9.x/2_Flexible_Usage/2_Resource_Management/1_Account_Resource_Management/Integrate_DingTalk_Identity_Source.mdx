## 功能说明

1. 支持 PC 和 Mac 终端用户使用手机钉钉扫描 TMA/TME 上的二维码进行身份验证和授权登录。
2. 支持使用钉钉帐号设置身份验证策略、业务资源授权策略、访问控制策略以及网络流量身份化。
3. TMC 支持手动或自动同步钉钉帐号信息，终端用户使用钉钉认证服务进行身份验证。
4. 钉钉不支持手机端其他应用通过钉钉 APP 关联认证，TM 手机端 APP 无法使用已登录的钉钉 APP 进行关联认证。

## 钉钉服务端配置

1. 使用管理员帐号登录[钉钉管理后台](https://oa.dingtalk.com)。
2. 点击【工作台】->【应用管理】，在"自建应用"区域创建应用。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_DingTalk_Identity_Source/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 选择"企业应用"，点击"快速创建"，选择"H5 微应用"，填写应用信息，开发方式选"企业自主开发"，点击"确定创建"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_DingTalk_Identity_Source/2.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

4. 在企业应用列表点击创建的应用，查看应用信息。记录三个应用凭证："AgentId"、"AppKey"、"AppSecret"，后续 TMC 配置时需要用到。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_DingTalk_Identity_Source/3.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

5. 选择"权限管理"菜单，为应用分配授权范围，建议选择"全部员工"。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_DingTalk_Identity_Source/4.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

开通权限：通讯录部门信息读权限、成员信息读权限、企业员工手机号信息、邮箱等个人信息、企业微应用后台免登接口的访问权限。

6. 选择"登录与分享"菜单，填入回调域名、iOS Bundle ID、Android 包名称、签名信息。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_DingTalk_Identity_Source/5.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

7. 回调地址在 TMC 的【策略管理】->【认证策略】->【认证接口】->【扫码认证】->【钉钉】配置页获取，"重定向地址"即回调地址。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_DingTalk_Identity_Source/5.5.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

"Android签名"填写固定值 **1f20c88c1f71f71517cc1e7d60c43279**，"应用包名"填写固定值 **com.yjx.flutter_yjx_trust**。

## 新建钉钉身份源

1. 选择菜单【策略管理】->【认证策略】，在页面顶部选择"认证接口"标签页。
2. 选择"扫码认证"->"钉钉"标签页。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_DingTalk_Identity_Source/7.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 填入"AgentID"、"AppKey"、"APPSecret"，点击"保存"。

## 设置钉钉帐号同步策略

1. 选择菜单【资源管理】->【帐号资源】，点击"帐号同步"按钮，选择"钉钉"标签页。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Integrate_DingTalk_Identity_Source/8.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 选择是否自动发送初始化邮件：开启则同步时自动发送到首次同步的帐号邮箱；关闭则需手动发送。
3. 同步方式：手动同步需每次点击"立即同步"；自动同步可设置间隔分钟数或定时同步。
4. 清空数据：清空从钉钉同步的所有帐号和帐号组信息。

完成配置后，点击"保存并立即同步"。
