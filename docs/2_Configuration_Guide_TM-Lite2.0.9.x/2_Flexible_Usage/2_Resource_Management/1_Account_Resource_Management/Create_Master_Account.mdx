## 功能说明

在 TMC 上配置两个以上不同身份源后，可为企业创建主帐号。主帐号定义详见[TM 的帐号分类](TM_Account_Classification#主帐号)。

## 创建主帐号

1. 选择菜单【资源管理】->【帐号资源】，点击顶部“主帐号管理”标签页。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Create_Master_Account/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 点击主帐号列表功能区的“多源身份关联”按钮。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Create_Master_Account/2.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 开启“身份关联”开关。
4. 选择关联策略：姓名、手机号、邮箱。TMC 将相同属性值的帐号聚合成一个主帐号，主帐号命名格式为 `属性1_属性2_属性3`。
5. 选择关联属性：指定主帐号其他属性值引用哪个身份源的对应值（如帐号组等）。
6. 点击“保存并启用”，TMC 开始自动生成主帐号。

## 注意事项

1. 主帐号不可新建、编辑和逐条删除。修改多源身份管理配置后，所有已生成的主帐号将被清空并按新策略重新生成。
2. 主帐号仅用于配置授权策略。主帐号变更后，基于其配置的授权策略和访问控制策略将重新计算。
3. 同一身份源存在同姓名多个帐号时，如仅用姓名单一属性关联，系统将随机取其中一个帐号的属性作为主帐号属性值。建议不要使用单一属性进行关联。
