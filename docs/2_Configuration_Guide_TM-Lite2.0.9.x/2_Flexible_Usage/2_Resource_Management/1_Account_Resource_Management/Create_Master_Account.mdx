## 功能说明

如果您已经在TMC上已经配置了两个以上的不同身份源后，您就可以为企业创建主帐号了。

关于主帐号的定义和用途，请参考：[TM 的帐号分类](../Account_Resource_Management/TM_Account_Classification#主帐号)。

## 创建主帐号

1. 选择菜单【资源管理】->【帐号资源】打开帐号管理页面，点击顶部“主帐号管理”标签页，进入主帐号管理页面，如下图所示：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Create_Master_Account/1.png" style={{width: "90%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

2. 点击主帐号列表功能区的“多源身份关联”按钮，显示多源身份关联配置的抽屉页，如下图所示：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/1_Account_Resource_Management/Create_Master_Account/2.png" style={{width: "70%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 开启“身份关联”开关。

4. 选择关联策略，可选项包括：姓名、手机号和邮箱。

  关联策略是指TMC在将多个身份源的帐号进行关联时，使用哪些帐号属性进行关联，如这些选中的属性值相等，则将帐号关联成一个主帐号，并将主帐号名称命名为“`属性1_属性2_属性3`”。

  例如：关联策略勾选了“姓名”、“手机号”，则不同身份源的帐号里，所有姓名与手机号都相同的帐号将被聚合成一个主帐号，主帐号名为“姓名_手机号”。

5. 选择关联属性

  关联属性是指：主帐号的其他属性值将引用哪一个身份源的对应属性值。

  例如：关联属性选择“企业微信”，则意味着生成的主帐号的其他属性值（如：帐号组等），将以企业微信的对应帐号属性值来赋值。

6. 点击多源身份关联配置页右下角的“保存并启用”按钮，TMC将开始自动生成主帐号。

## 注意事项

1. 主帐号为TMC自动计算生成，因此主帐号不可新建、编辑和逐条删除，每次修改多源身份管理配置，所有已生成的主帐号将被清空，并按照新的策略重新生成。

2. 主帐号仅能用于配置授权策略，如您之前基于主帐号配置了授权策略，当主帐号出现变更后，授权策略将发生变化，细粒度的访问控制策略将重新计算。

3. 当同一身份源存在同样姓名的多个帐号，且在关联身份时选择用姓名单一属性进行关联，关联属性也是选择的这个身份源时，系统将随机取多个帐号中的其中一个的属性作为主帐号属性值。

  建议当同身份源有姓名相同，或手机号，或邮箱相同时，不要使用单一属性进行关联。
