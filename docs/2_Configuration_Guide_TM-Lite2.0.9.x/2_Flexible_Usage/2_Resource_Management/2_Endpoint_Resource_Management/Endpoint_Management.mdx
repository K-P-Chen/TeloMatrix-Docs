## 功能说明

在 TM 安全云网中，"帐号+终端"代表一个独立的网络实体，可作为被授权的网络对象。同一帐号使用不同终端、或不同帐号使用同一终端，均视为不同的网络实体，授权范围可完全不同。

终端加入 TM 安全云网时需绑定一个帐号进行注册。注册时可将终端用途设为以下四种类型：

- **个人终端：**用户的私人终端，注册后不可再注册为其他帐号的个人、办公或公共终端，只能被注册为临时终端。
- **办公终端：**用户专用办公终端，约束同上。
- **临时终端：**已被注册的终端用其他帐号登录时，需注册为临时终端。临时终端有强制有效时间，超期后自动注销。
- **公共终端：**由管理员设置，需输入超级管理员（sysadmin）密码授权。多个帐号可共用而无需逐一注册。

## 完整的终端注册流程

此流程为最严格的新终端管控方式，需管理员审批确认。

1. **下载安装 TM 客户端APP：**终端用户使用有效 TM 帐号进行首次认证登录。
2. **终端用户提交注册申请：**TMA 识别未注册终端后弹出注册页面，填写终端名称、用途、申请理由。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/2_Endpoint_Resource_Management/Endpoint_Management/1.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. **管理员在 TMC 审批：**在消息中心查看终端注册申请，可审批通过（可修改终端名称、用途、分配终端组）或驳回（填入理由，通过邮件通知）。

## <span id="简化的终端注册流程">简化的终端注册流程</span>

简化流程可在大范围部署 TMA 时提升效率。重保期间可恢复强管控流程。

### 方式一：终端自动注册

选择菜单【策略管理】->【全局策略】，在"终端全局配置"区域关闭终端注册流程。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/2_Endpoint_Resource_Management/Endpoint_Management/4.png" style={{width: "60%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

关闭后，TMA 自动提交终端注册申请，TMC 自动审批通过。终端名称使用操作系统名称，终端用途使用默认设置，终端分配到默认终端组。

### 方式二：开启自动审批

选择菜单【策略管理】->【全局策略】，在"终端全局配置"区域开启终端自动审批流程。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/2_Endpoint_Resource_Management/Endpoint_Management/5.png" style={{width: "40%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

终端用户提交注册申请后，TMC 立即自动审批通过，无需管理员操作。

### 方式三：开启短信验证审批

选择菜单【策略管理】->【全局策略】，在"终端全局配置"区域开启终端自动审批流程。

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/2_Resource_Management/2_Endpoint_Resource_Management/Endpoint_Management/6.png" style={{width: "80%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

TMC 自动发送 6 位验证码到登录帐号的手机上，用户输入正确验证码后自动完成注册。使用前请确认已配置短信网关。
