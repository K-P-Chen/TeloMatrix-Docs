## 功能说明

授权策略因子是预定义的 URI 列表或 SSH 指令集，在配置授权策略时引用，作为应用层访问的 URI 黑名单和指令黑名单。

在授权策略配置页面，点击"授权策略因子"打开配置标签页：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/3_Policy_Management/Authorization_Policy_Factors/1.png" style={{width: "100%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

## 配置 URI 列表

1. 在左侧策略因子类型栏选择"URI列表"，右侧显示已配置的 URI 列表。
2. 点击"新建"打开 URI 列表配置页面：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/3_Policy_Management/Authorization_Policy_Factors/2.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 配置项说明：
   - **名称：**必填。
   - **备注：**非必填，最多200字符。
   - **URI：**每行一条 URI，以回车键隔开，支持通配符。输入后需点击"新增"按钮才会录入到列表。
4. 点击"保存"完成创建，在配置授权策略时可引用。

## 配置 SSH 指令集

1. 在左侧策略因子类型栏选择"SSH指令集"，右侧显示已配置的 SSH 指令集。
2. 点击"新建"打开 SSH 指令集配置页面：

<img src="/images/2_Configuration_Guide_TM-Lite2.0.9.x/2_Flexible_Usage/3_Policy_Management/Authorization_Policy_Factors/3.png" style={{width: "50%", display: "block", margin: "0 auto", border: "1px solid #e5e7eb", boxShadow: "0 2px 8px rgba(0,0,0,0.08)"}} />

3. 配置项说明：
   - **名称：**必填。
   - **备注：**非必填，最多200字符。
   - **指令：**每行一条 SSH 指令，以回车键隔开，支持通配符。输入后需点击"新增"按钮才会录入到指令集。
4. 点击"保存"完成创建，在配置授权策略时可引用。
